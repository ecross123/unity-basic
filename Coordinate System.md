世界坐标系
-
原点为世界的中心点，轴向对应前方（z），上方（y），右方（x），方向是固定的
```csharp
相对世界坐标系变化
this.transform.position;//位置
this.transform.rotation;//旋转
this.transform.eulerAngles;//欧拉角
this.transform.lossyScale;//缩放大小
```
相对坐标系
-
原点对应物体本身的中心点，三个轴向z，y，x分别为物体的前方上方和右方
```csharp
相对父对象坐标系变化
this.transform.localPosition;//位置
this.transform.localRotation;//旋转
this.transform.localEulerAngles;//欧拉角
this.transform.localScale;//缩放大小
```
屏幕坐标系
-
原点为屏幕左下角，向右为x正方向，向上为y正方向，最大宽高分别为Screen.width/Screen.height
```csharp
Input.mousePosition;//鼠标坐标
Screen.wedth;//屏幕宽
Screen.height;//屏幕高
```
视口坐标系
-
原点为屏幕左下角，向右为x正方向，向上为y正方向，左下角为（0，0），右上角为（1，1），将坐标进行了单位化
和camera的视口范围相关

坐标的转换
-
```csharp
//世界转本地
this.transform.InverseTransformDirection;
this.transform.InverseTransformPoint;
this.transform.InverseTransformVector;
//本地转世界
this.transform.TransformDirection;
this.transform.TransformPoint;
this.transform.TransformVector;
//世界转屏幕
Camera.main.WorldToScreenPoint;
//屏幕转世界
Camera.main.ScreenToWorldPoint;
//世界转视口
Camera.main.WorldToViewportPoint;
//视口转世界
Camera.main.ViewportToWorldPoint;
//视口转屏幕
Camera.main.ViewPortToScreenPoint;
//屏幕转视口
Camera.main.ScreenPointToViewPort;
```
